  <summary>Quickstart</summary>

```shell
# Clone the repository.
git clone git@github.com:doraboateng/api.git
cd api

# Run the database migrations. This will launch the API locally, build
# the base image and run the migration files.
./run migrate

# Launch your IDE.
code .
```
